# UTMForms
Работа с утм с GUI
В возможностях
1 - асинхронная перезагрузка службы Transport по IP машины
2 - ребут ПК
3 - список УТМ обновляемый из TXT файла
4 - chromium для проверки страницы утм
5 - проверка по команде ping доступности пк с утм
